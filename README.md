Cyclomatic complexity 

Цикломатската комплексност ја пресметав според формулата V(G)=E-N+2, каде што E означува број на ребра, а N број на јазли.

N=27

E=35

V(G)=E-N+2=35-27+2=10 

Цикломатската комплексност изнесува 10.
